# Text-Analyser
Java console app that analyzes input text: word count, letter statistics, longest word, and frequency analysis

---

## 🚀 Features

The program analyzes the entered text and outputs:

- Total number of characters  
- Characters without whitespace  
- Number of words  
- Number of vowels  
- Number of consonants  
- Number of digits  
- Longest word  
- Most frequent letter  
- Average word length  

---

## 🧠 How It Works

The program:

1. Prompts the user to enter a text
2. Validates the input (prevents empty input)
3. Processes the text using multiple methods
4. Outputs the analysis results
5. Allows the user to restart the program
